# LPM Comparator

## Python interpreter
Select interpreter in VSCode for python. Select "Select interpreter" and navigate to the python.exe in the folder /flask-server/venv.

## Starting server
Navigate to correct folder:
cd flask-server

Activate virtual environment:
Windows: .\venv\Scripts\activate
Linux: source venv/bin/activate

Start the server:
Windows: python.exe server.py

## Starting client
Open another terminal and type:
cd client
npm start

This starts the development server.
